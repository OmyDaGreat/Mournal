# Mournal

This is a journaling app built on top of the [DailyMalefic](https://github.com/OmyDaGreat/DailyMalefic) REST server! It's built with [Kobweb](https://kobweb.varabyte.com/), a cool Kotlin web development framework. The server is designed to be used by one person and shared with others, though it can totally go beyond that because of the server's immense capabilities.
